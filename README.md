# Mini CRM (Client Lead Management System)

A simple full-stack CRM application built using the MERN stack (MongoDB, Express, React, Node.js).  
It allows an admin to manage client leads, track their status, and view basic analytics.

##  Features

- Admin login authentication (JWT)
- Add, edit, and delete leads
- Track lead status (New, Contacted, Converted)
- Search leads
- Simple analytics dashboard with chart
- Responsive design (mobile friendly)

---

## Tech Stack

- React (Vite)
- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- Chart.js
